# Audio Samples

Place the following OGG files in this folder before building:

| File | Drum hit |
|------|----------|
| `Kick1.ogg` | Bass drum |
| `Kick2.ogg` | Bass drum 2 (double pedal) |
| `Snare1.ogg` | Snare drum |
| `HhClosed.ogg` | Closed hi-hat |
| `Ride1.ogg` | Ride cymbal |
| `Crsh1.ogg` | Crash cymbal 1 |
| `Crsh2.ogg` | Crash cymbal 2 |
| `HiMidTom.ogg` | Hi/mid tom |
| `LoMidTom.ogg` | Low/mid tom |
| `LowFlrTom.ogg` | Low floor tom |

All files must be OGG Vorbis format. Free CC0 sample packs such as those on
[freesound.org](https://freesound.org) are a good source.
